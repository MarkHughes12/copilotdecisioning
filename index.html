Copilot Studio: Agent Decisioning Guide

Follow this guide to find the right agent creation tool for your AI development needs.

START HERE

QUESTION 1

Is your primary goal to create an agent that answers questions based only on content from a single SharePoint site or specific libraries/folders?

➡️ YES

RECOMMENDATION

SharePoint Agent

The quickest way to create a focused Q&A experience for a team or project within SharePoint. This is ideal for simple, context-specific knowledge management.

⬇️ NO

QUESTION 2

Do you want to build a simple agent for personal or team use (e.g., summaries, Q&A from various sources) with a no-code, natural language interface?

➡️ YES

RECOMMENDATION

Agent Builder

A user-friendly, no-code tool in Copilot Studio that lets business users create custom agents for personal or team productivity without deep technical expertise.

⬇️ NO

QUESTION 3

Do you need a robust, customized agent with requirements like enterprise data connections, complex workflows, or a custom UI?

➡️ YES

RECOMMENDATION

Copilot Studio

A powerful low-code platform for demanding use cases. Use this for integrations, complex conversations, and granular control over your agent's functionality.

⬇️ NO

QUESTION 4

Does your process require multiple, specialized agents to collaborate and delegate tasks to solve a complex, cross-functional problem?

➡️ YES

RECOMMENDATION

Multi-Agent Orchestration

An advanced feature within Copilot Studio that allows you to build a team of collaborating agents for complex, enterprise-level challenges.

➡️ NO

RECOMMENDATION

Copilot Studio

If you've reached this point, your needs can likely be met by the core Copilot Studio platform. It offers a broad range of capabilities for most single-agent scenarios.
